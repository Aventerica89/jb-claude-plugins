# JB Claude Plugins

Claude Code plugin marketplace for JBMD Creations.

## Install

```bash
claude plugin marketplace add Aventerica89/jb-claude-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| [1p-local-auth](https://github.com/Aventerica89/1p-local-auth) | Manage local dev OAuth credentials via 1Password |

## Install a Plugin

```bash
claude plugin install 1p-local-auth@jb-claude-plugins
```
